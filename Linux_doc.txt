----------
LINUX/UNIX is an operating system by using OS users or applications can communicate with 
hardware componets
Features of LINUX OS:
----------------------
1>Linux is FOSS ("Freeware and Open source Software")
2>Linux is multi user OS
3>It is user friendly .It provides both CUI (charcter user interface) and GUI (Graphical User Interface)

Types of files in linux:
-------------------------
We know everything in python treated as Object

similarly in Linux every thing treated as file

There are three types of files in Linux

1.Normal or ordinary file
2.Directory file
3.Device file


1.Normal or ordinary file:
---------------------------
These files contains only data it can be either test file (like abc.txt) or binary files
(like audio,images,video files)

2.Directory file:
-----------------
These files represents directories
In windows we can use folder terminalogy .Where in linux we can use directory terminalogy
Directory contains files and sub directories

3.Device file:
-------------
Every device treated as file only .By using these files we can communicate with that device.

Eg:
	Terminal also one device in linux OS so terminal also a file


1st terminal:
jagang@jagang-VirtualBox:~$ tty
/dev/pts/0                     --------------->0 represents first terminal file
jagang@jagang-VirtualBox:~$ ^C

2nd terminal:
jagang@jagang-VirtualBox:~$ tty
/dev/pts/1                        --------------->1 represents 2nd terminal file




eg:
terminal 1:
jagang@jagang-VirtualBox:~$ cat >/dev/pts/1
Hello
How are you
i am fine 


terminal 2:
jagang@jagang-VirtualBox:~$ Hello
How are you
i am fine

6.mnt:
---------

mounting means the files in external devices are like pendrive,cd ...e.t.c  
attached to linux file system.Then only we can use external files.This attchment process is
known as mounting.

There are two types of mounting process are there

1.manual mounting
2.automatic mounting

In old versions of Linux os we can perform mounting process manually.
In manual mounting process all the files added to mnt directory


In new versions of Linux os  mounting process automatically done.In automatic mounting process
the files in external devices stored media directory
eg:
